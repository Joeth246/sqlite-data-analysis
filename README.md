# SQLite Data Analysis Project

## Overview
This project demonstrates structured data analysis using SQLite and SQL queries to extract, filter, and analyze information from relational datasets.

The analysis focused on querying database tables, identifying relationships between datasets, and using aggregation techniques to generate meaningful insights from structured data.

## Objectives
- Query and analyze relational datasets using SQL
- Filter and organize structured data
- Use aggregation and grouping functions for analysis
- Explore relationships between database tables
- Develop foundational database analysis skills

## Environment
- SQLite
- SQL query language
- Relational database tables
- Structured sample datasets

## Analysis Performed

### Data Exploration
- Queried database tables to review dataset structure and contents
- Inspected records and field relationships across tables

### Filtering and Selection
- Applied filtering conditions using `WHERE`
- Retrieved relevant subsets of data for targeted analysis

### Aggregation and Grouping
- Used `COUNT`, `GROUP BY`, and `ORDER BY`
- Identified trends and activity patterns within datasets

### Relational Analysis
- Used SQL joins to combine related data across tables
- Examined relationships between entities and records
